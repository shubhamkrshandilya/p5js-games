# 🎮 Creative Coding Retro Games Arcade

Welcome to the **Retro Games Arcade** repository! This is a collection of globally famous retro puzzle and board games remade with modern fluid animations, glassmorphism aesthetics, neon color guides, and full mobile-responsive touch controller overlays.

---

## 🕹️ Active Games Directory

### 1. 🧱 [Neon Tetris](https://shubhamkrshandilya.github.io/p5js-games/tetris/)
* **Scientific Focus**: Rigid body collision matrices, translation kinematics, state queues, and particle physics.
* **Key Features**:
  - Full mobile viewport layouts with dynamic touch controller keys.
  - **Ghost Drop Projection**: A real-time overlay showing where the piece will land.
  - Hold swap mechanism and next piece queue previews.
  - Line-clear score multipliers and speed increments.
  - Particle vector bursts upon clearing rows.

## 🛠️ Local Development & Execution

To host this repository locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/shubhamkrshandilya/p5js-games.git
   cd p5js-games
   ```

2. **Launch a local server**:
   * Using Python 3:
     ```bash
     python3 -m http.server 8082
     ```
   * Using Node.js:
     ```bash
     npx live-server --port=8082
     ```

3. Open **`http://localhost:8082`** in your browser.
